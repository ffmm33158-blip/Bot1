# 🤖 بوت تنظيم الملاحظات الذكي

مساعدك الشخصي الذكي في تنظيم حياتك اليومية عبر تليجرام

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)

[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 📝 الوصف

بوت ذكي لتنظيم وإدارة الملاحظات الشخصية عبر تليجرام، يتميز بنظام تصنيف متقدم وتذكيرات دقيقة مع واجهة أزرار تفاعلية سهلة الاستخدام.

## 🌟 المميزات الرئيسية

### 📝 إدارة الملاحظات
- إضافة ملاحظات جديدة بسهولة
- تنظيم الملاحظات في تصنيفات مخصصة
- عرض منظم للملاحظات مجمعة حسب التصنيف
- تعديل وحذف الملاحظات

### 🎯 نظام الأولويات الملون
- 🔴 **مهم جداً**: للمهام العاجلة والحرجة
- 🟡 **مهم**: للمهام المتوسطة الأهمية  
- 🟢 **عادي**: للملاحظات والأفكار العامة

### ⏰ نظام التذكيرات الذكي
#### خيارات سريعة:
- بعد 30 دقيقة، ساعة، ساعتين، 6 ساعات
- غداً 9 صباحاً، غداً 6 مساءً
- بدون تذكير

#### نظام الوقت المخصص (4 خطوات دقيقة):
1. اختيار اليوم: اليوم، غداً، بعد غد، الأسبوع القادم
2. اختيار الساعة: من 00-23 (24 ساعة)
3. اختيار مجموعة الدقائق: 00-09, 10-19, 20-29, 30-39, 40-49, 50-59
4. اختيار الدقيقة الدقيقة: أي دقيقة من 00-59

### 📁 نظام التصنيفات المرن
- تصنيفات افتراضية: عام، مهام، أفكار
- إضافة تصنيفات مخصصة لا نهائية
- تعديل أسماء التصنيفات
- حذف التصنيفات مع نقل الملاحظات لـ "عام"

### 🔍 البحث المتقدم
- البحث في العنوان والنص والتصنيف
- عرض أول 10 نتائج مع معاينة
- تنسيق مرقم مع عداد النتائج

### 📊 الإحصائيات التفصيلية
- إجمالي الملاحظات والتصنيفات
- الملاحظات الحديثة (آخر 7 أيام)
- تفصيل كل تصنيف مع عدد الملاحظات
- توزيع الأولويات مع الإحصائيات

### 💾 النسخ الاحتياطية
- إنشاء نسخة احتياطية كاملة بصيغة .txt
- تنظيم حسب التصنيفات
- معلومات شاملة لكل ملاحظة
- اسم الملف: `notes_backup_YYYYMMDD_HHMMSS.txt`

### 🎨 واجهة تفاعلية متقدمة
- أزرار منظمة وواضحة لكل وظيفة
- نظام التنقل بين الصفحات (5 عناصر لكل صفحة)
- أزرار الإلغاء في كل مرحلة
- معاينة فورية للاختيارات

## 🚀 الأوامر الثمانية الكاملة

### 1. `/start` 🚀
**الوظيفة**: تنشيط وتفعيل البوت  
**النتيجة**: رسالة ترحيب شاملة مع شرح جميع الأوامر والمميزات

### 2. `/add` ➕
**الوظيفة**: إضافة ملاحظة جديدة أو تصنيف جديد  
**العملية**: 6 خطوات متدرجة لإضافة ملاحظة كاملة

### 3. `/notes` 📚
**الوظيفة**: عرض جميع الملاحظات السابقة منظمة  
**التنسيق**: مجمعة حسب التصنيفات مع عرض أول 5 ملاحظات لكل تصنيف

### 4. `/edit` ✏️
**الوظيفة**: تعديل شامل للملاحظات والتصنيفات  
**الخيارات**: تعديل العنوان، النص، الأولوية، التذكير، أو حذف الملاحظة

### 5. `/search` 🔍
**الوظيفة**: البحث المتقدم في جميع الملاحظات  
**النطاق**: العنوان + النص + التصنيف

### 6. `/stats` 📊
**الوظيفة**: إحصائيات تفصيلية شاملة  
**المعلومات**: إجمالي البيانات + التفصيل + توزيع الأولويات

### 7. `/backup` 💾
**الوظيفة**: إنشاء نسخة احتياطية كاملة  
**التنسيق**: ملف نصي منظم .txt مع جميع التفاصيل

### 8. `/menu` 📋
**الوظيفة**: عرض قائمة شاملة بجميع الأوامر  
**المحتوى**: وصف مختصر لكل أمر + نصائح الاستخدام

## 🛠️ التثبيت والإعداد

### المتطلبات
- Python 3.8+
- حساب تليجرام
- بوت تليجرام (من @BotFather)

### 1. تحميل الملفات
```bash
git clone https://github.com/your-username/notes-bot.git
cd notes-bot
```

### 2. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 3. إعداد توكن البوت
```bash
# في متغيرات البيئة
export BOT_TOKEN="YOUR_BOT_TOKEN_HERE"

# أو في ملف .env
BOT_TOKEN=YOUR_BOT_TOKEN_HERE
```

### 4. تشغيل البوت
```bash
python main.py
```

## 🔧 التخصيص والتطوير

## 📞 الدعم والمساعدة

### المشاكل الشائعة

**البوت لا يعمل:**
- تأكد من صحة توكن البوت
- تحقق من اتصال الإنترنت
- راجع ملف `bot.log` للأخطاء

**البيانات لا تحفظ:**
- تحقق من أذونات الكتابة في المجلد
- تأكد من وجود مساحة كافية على القرص

**التذكيرات لا تعمل:**
- تأكد من تشغيل نظام التذكيرات
- تحقق من الوقت والمنطقة الزمنية

### تقرير المشاكل
إذا واجهت أي مشكلة، يرجى إنشاء Issue على GitHub مع:
- وصف المشكلة
- خطوات إعادة الإنتاج
- محتوى ملف السجل `bot.log`
